C++ programmēšanas kursi Rīgā - tagad apgūsti C++ arī Latvijas galvaspilsētā!

Kļūsti par C++ izstrādātāju 6 mēnešos mūsu kursu centrā Rīgā!

Mūsu C++ programmēšanas kursi ir paredzēti tiem, kas vēlas ātri un efektīvi apgūt profesionālu C++ programmēšanu. Piedāvātās apmācības notiek Rīgā, viena no modernākajām pilsētām Baltijā, sagatavo dalībniekus darbam spēļu izstrādes studijās, embedded sistēmu uzņēmumos un augstas veiktspējas programmatūras izstrādē. Studiju laikā Tu iegūsi praktiskas prasmes un dziļāku izpratni par C++ valodu - sākot no pamatiem līdz sarežģītākām tēmām. Kursā izmantosim reālus projektus un uzdevumus, lai Tu spētu uzreiz pielietot zināšanas praksē. Šie C++ kursi būs lielisks pamats tavam IT portfolio!

  • Kursa ilgums: 24 nedēļas (96 akadēmiskās stundas)
  • Klase: 2 reizes nedēļā (vakariem 18:00-21:00)
  • Nodarbību veids: klātienes nodarbības
  • Vieta: "Kodiņš", kursu centrs "Rīga" (Raiņa Bulvāris 13, Rīga).
  • Grupas lielums: Maksimums 10 dalībnieki
  • Cena: 1200€ (vai 4 maksājumi pa 300€)
Rezervēt vietu

Programma

1. C++ Pamati un Atmiņas Pārvaldība

  • Pointeri un references
  • Stack vs heap atmiņa
  • RAII un smart pointeri
  • STL konteineri un algoritmi

2. Objektorientēta Programmēšana C++

  • Klases un inheritance
  • Polimorfisms un virtual funkcijas
  • Move semantika
  • Design patterns (Factory, Observer)

3. Daudzpavedienu Programmēšana

  • Thread un mutex
  • Race conditions un deadlocks
  • Async programmēšana
  • Lock-free algoritmi

4. Spēļu Izstrāde ar Unreal Engine

  • C++ un Blueprint integrācija
  • Game loop un fizika
  • AI pamati
  • Optimizācijas paņēmieni

Kas ietverts C++ programmēšanas kursā?

Šis C++ kurss Rīgā piedāvā praktiskas iemaņas, kas būs noderīgas gan iesācējiem, gan tiem, kuri vēlas padziļināt savas zināšanas programmēšanā un sistēmu izstrādē.

  • 3 pilnvērtīgi projekti portfolio (spēle, multithread aplikācija, sistēmu programmatūra)
  • Mentoru sesijas ar nozares ekspertiem
  • Git versiju kontroles sistēma no pirmās nodarbības
  • Karjeras diena ar vadošajiem Latvijas tehnoloģiju uzņēmumiem
  • Mācību materiāli un C++ kodu piemēriem arī pēc kursa beigām
