C++ programmēšana

Kļūsti par C++ izstrādātāju 6 mēnešos

Intensīvs kurss, kas sagatavos tevi darbam spēļu studijās, embedded sistēmu uzņēmumos un augstas veiktspējas programmatūras izstrādē. Studiju laikā tu iegūsi praktiskas prasmes, kas nepieciešamas reālās darba situācijās, un attīstīsi savas zināšanas, sākot no pamatiem līdz sarežģītākām tēmām. Nodarbībās tiks izmantoti reāli piemēri, uzdevumi un projekti, kas palīdzēs labāk izprast C++ valodas iespējas un pielietojumu dažādās nozarēs. Noslēgumā tu būsi izstrādājis vairākus projektus, kas kalpos kā labs pamats tavam portfolio.

  • Kursa ilgums: 24 nedēļas (96 akadēmiskās stundas)
  • Klase: 2 reizes nedēļā (vakariem 18:00–21:00)
  • Grupas lielums: Maksimums 10 dalībnieki
  • Cena: 1200€ (vai 4 maksājumi pa 300€)
Rezervēt vietu

Programma

1. C++ Pamati un Atmiņas Pārvaldība

  • Pointeri un references
  • Stack vs heap atmiņa
  • RAII un smart pointeri
  • STL konteineri un algoritmi

2. Objektorientēta Programmēšana C++

  • Klases un inheritance
  • Polimorfisms un virtual funkcijas
  • Move semantika
  • Design patterns (Factory, Observer)

3. Daudzpavedienu Programmēšana

  • Thread un mutex
  • Race conditions un deadlocks
  • Async programmēšana
  • Lock-free algoritmi

4. Spēļu Izstrāde ar Unreal Engine

  • C++ un Blueprint integrācija
  • Game loop un fizika
  • AI pamati
  • Optimizācijas paņēmieni

Kas ietverts kursā?

  • 3 pilnvērtīgi projekti portfolio (spēle, multithread aplikācija, sistēmu programmatūra)
  • Mentoru sesijas ar nozares ekspertiem
  • Git versiju kontroles sistēma no pirmās nodarbības
  • Karjeras diena ar vadošajiem Latvijas tech uzņēmumiem
  • Mācību materiāli un kodu piemēri pēc kursa
