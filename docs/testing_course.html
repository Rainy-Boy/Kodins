Testēšanas apmācības

Kļūsti par kvalificētu testētāju 3 mēnešos

Praktisks kurss, kas sagatavos tevi darbam kā QA inženieris vietējos un starptautiskos IT uzņēmumos.

  • Kursa ilgums: 12 nedēļas (60 akadēmiskās stundas)
  • Klase: 2 reizes nedēļā (vakariem 18:00-21:00)
  • Grupas lielums: Maksimums 15 dalībnieki
  • Cena: 750€ (vai 3 maksājumi pa 275€)
Pieteikties bezmaksas konsultācijai

Kursa saturs

1. Testēšanas Pamati

  • QA inženiera loma izstrādes procesā
  • Testēšanas veidi un līmeņi
  • Testa gadījumu rakstīšana
  • Defektu dokumentēšana (bug reports)

2. Manuālā Testēšana

  • Funkcionālā testēšana
  • Lietojamības testēšana
  • Testēšana dažādās platformās
  • Testa plānu izstrāde

3. Automatizētā Testēšana

  • Selenium WebDriver pamati
  • Testu automatizācija ar Java/Python
  • API testēšana ar Postman
  • Testu izpildes rāmji

4. Paplašinātas QA Prakses

  • Veiktspējas testēšana
  • Drošības testēšana
  • CI/CD ieviešana testēšanā
  • Agile testēšanas pieeja

Kāpēc izvēlēties mūsu QA kursu?

  • Reāli testēšanas scenāriji - Strādāsi ar pašreizējiem uzņēmumu produktiem
  • Portfolio projekti - Testa komplekti un automatizācijas skripti
  • Karjeras konsultācijas - CV un LinkedIn profila uzlabošana
  • Partneru uzņēmumu prakse - Iespēja iegūt reālu pieredzi
